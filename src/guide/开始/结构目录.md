# Yunzai 结构主要目录
> 这里只显示主要结构，其他目录此处不做展示
---
``` md
.Yunzai
 ├─ .git
 ├─ config //配置
 │  ├─ config
 │  │  ├─ bot.yaml //bot配置
 │  │  ├─ group.yaml //群组配置
 │  │  ├─ other.yaml //其他配置，比如主人设置，黑名单之类的
 │  │  ├─ redis.yaml //redis配置，不用管
 │  └─ default_config //作为config的备份文件，千万不要动
 │
 ├─ data //数据
 │  ├─ face //群组表情存放位置
 │  ├─ myscookie //cookie存放位置
 │  └─ userdata //绑定uid存放位置
 └─ plugins
     ├─ example //js插件存放位置
     ├─ genshin //自带插件，别再找不到米游社资讯推送了
     ├─ other //小白勿动
     ├─ system //同上
     ├─ …… //其他插件依次类推

```
